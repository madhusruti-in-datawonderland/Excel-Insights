This project focuses on building an interactive dashboard to analyze Amazon sales data and generate meaningful business insights.

# DATASET USED :- https://github.com/madhusruti-in-datawonderland/Excel-Insights/blob/main/Mini%20Project%203%20Dataset.xlsx

📌 PROJECT NAME :- Amazon Workforce & Sales Intelligence Dashboard

🧩 THE PROBLEM :- 
The sales team was working with a dataset of 100 Amazon retail transactions spanning multiple regions, product categories, and payment methods. However, the raw data was unstructured and difficult to interpret, making it challenging to track sales performance, analyze delivery efficiency, and identify cancellation trends.

🎯 THE GOAL :- 
To transform this raw dataset into a clear, interactive Excel dashboard that enables quick analysis, uncovers key insights, and supports better decision-making.

🧠 MY APPROACH :-
1️⃣ Data Cleaning & Pre-processing :
• Converted all date columns into proper date format for better analysis
• Cleaned the dataset by removing extra spaces and handling blank values
• Standardized key fields like Region, Payment Method, and Delivery Status to maintain consistency
2️⃣ Applying XLOOKUP for Cross-Referencing :
• Built a Customer Master dataset
• Applied XLOOKUP to map Customer IDs to Customer Names for improved reporting
3️⃣ Creating Pivot Tables :
• Analyzed total revenue across regions and product categories
• Calculated order counts based on delivery status
• Determined average delivery time using order and delivery dates
4️⃣ Advanced Reporting : KPI Metrics & Calculated Fields : 
Developed key performance indicators to assess business performance :
• 💰 Total revenue generated
• ⚡ Fast delivery rate (orders delivered within 2 days)
• 🏆 Top-performing regions
• 📦 Best-selling product category
• ❌ Cancellation rate by region
5️⃣ Final Dashboard Design :
Designed an interactive Excel dashboard featuring:
• Pivot charts for visualization
• Slicers for dynamic filtering (Region, Payment Method, Delivery Status)

📈 OUTCOME :-
The dashboard enables decision-makers to quickly:
✔ Analyze sales performance across different regions
✔ Recognize top-performing products
✔ Monitor delivery performance and speed
✔ Identify patterns and trends in order cancellations
